What is a Chi-square Test?

The chi-square test is a statistical method used to determine if there is a significant association between categorical variables. Common applications include:

  • Independence: Testing if two categorical variables are related.
  • Homogeneity: Comparing distributions across groups.
  • Goodness of Fit: Checking if observed data matches expected proportions.
